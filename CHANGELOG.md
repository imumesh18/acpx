# Changelog

All notable changes to this project will be documented in this file.

## [0.0.0]

- Initial repo baseline for an ACP-focused Rust library project.
- Repository scaffolding, docs, CI, supply-chain audit, local `just` and
  `devenv` tooling, `.ref` reference-code workflow, and `git-cliff`-driven
  release automation.
- No stable public ACP API is committed yet.
