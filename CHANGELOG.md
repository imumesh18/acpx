## [0.1.0] - 2026-03-15


### Added
- Runtime and fixture-agent foundations for deterministic ACP subprocess tests and examples.

- A subprocess-backed `Connection` wrapper for ACP over stdio, including request forwarding and captured `session/update` notifications.

- A handwritten `AgentServer` launch contract with `CommandAgentServer` and `CommandSpec` for fixed local launch commands.

- A generated `agent_servers` catalog for the committed ACP registry snapshot.

- Curated registry lookup helpers for resolving official ACP server ids and host-platform binary metadata.

- A CLI example that exercises one-shot agent sessions against manual and registry-backed server definitions, plus tighter registry naming.



### Fixed
- GitHub Actions tool installation now works with current `cargo-binstall` releases by avoiding duplicate confirmation flags.



### Changed
- Quality gates, formatting, and release automation were unified around the current Rust and docs toolchain.

- The ACP spec, roadmap, and repository guardrails were aligned with the shipped library surface.

- The generated agent-server catalog was consolidated behind a cleaner public module boundary.

- Product docs and maintainer docs were refreshed to match the shipped `0.1.0` API surface.

- The README now surfaces CI, crates.io, docs.rs, MSRV, and license badges for release visibility.

## [0.0.1] - 2026-03-15
